# PWA Assets Needed

## Required Images

### 1. App Icon (icon-512.png)
- **Size**: 512x512 pixels
- **Format**: PNG
- **Background**: #1a5f3a (donkergroen)
- **Design**: Clean moskee of boek icoon in wit/goud
- **Style**: Minimalistisch, professioneel

### 2. Favicon (favicon.png)
- **Size**: 192x192 pixels
- **Format**: PNG
- **Design**: Same as app icon, scaled down
- **Transparency**: Supported

### 3. Splash Screen (splash.png)
- **Size**: 1242x2436 pixels (iPhone X+ ratio)
- **Format**: PNG
- **Background**: #1a5f3a (donkergroen)
- **Content**: Centered logo + "Ruqiyah voor Kinderen" text
- **Style**: Clean en professioneel

## Design Guidelines
- **Primary Color**: #1a5f3a
- **Secondary Color**: #0d3d1f
- **Accent Color**: #e8f5e9
- **Text Color**: White
- **Font Family**: Clean, readable sans-serif

## Tools for Creation
- Canva (recommended)
- Figma
- Adobe Illustrator
- Photoshop

## Icon Ideas
- Open book with Arabic text
- Mosque silhouette
- Prayer beads (tasbih)
- Combination of book + mosque

## File Naming Convention
- Keep exact names as specified
- Use lowercase with hyphens
- Ensure .png extension

Once created, place these files in the `/public` directory and remove this README file.
