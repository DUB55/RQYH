# Ruqiyah voor Kinderen PWA

Een Progressive Web App voor het lezen en beluisteren van islamitische smeekbeden (ruqiyah) voor kinderen.

## Features
- ✅ 6 authentieke ruqiyah smeekbeden
- ✅ Arabische tekst met harakaat
- ✅ Nederlandse vertaling
- ✅ Fonetiche uitspraak
- ✅ Audio afspelen
- ✅ Werkt offline (beperkt)
- ✅ iOS PWA support

## Installatie
```bash
npm install
npx expo start
```

## Build
```bash
npx expo export -p web
```

## Project Structuur
- `src/components/` - React Native components
- `src/data/` - Smeekbeden data
- `src/hooks/` - Custom hooks
- `src/types/` - TypeScript types
- `src/styles/` - Theme en styling
- `public/` - PWA assets
